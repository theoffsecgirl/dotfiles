# Python venv template

Template base para proyectos Python en bug bounty.

## Uso

```bash
cd ~/hunting/scripts/mi-proyecto
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Dependencias base

- **requests** - HTTP tradicional
- **httpx** - HTTP async moderno
- **python-dotenv** - Variables de entorno

Añade más según necesites.
