
Este repositorio contiene mi configuración personalizada para el terminal en Linux Mint, usando Zsh, Tilix y Powerlevel10k con estilo rosas 🦂.

---

## ✨ Características

- Prompt moderno con Powerlevel10k, colores pastel y fondo oscuro.
- Icono de escorpión en el prompt para darle personalidad.
- Configuración modular con alias, funciones y variables en archivos separados.
- Tilix configurado con paleta de colores rosa.
- Fuente recomendada: Hack Nerd Font Mono.

## 🛠 Instalación rápida

Clona el repositorio:

```bash
git clone https://github.com/theoffsecgirl/dotfiles-rosepunk.git ~/dotfiles
```
Entra en la carpeta y ejecuta el script de instalación:

```bash
cd ~/dotfiles
chmod +x install.sh
./install.sh
source ~/.zshrc
```
