# Dotfiles de theoffsecgirl

Estos son mis **dotfiles** personales para configurar entornos y terminales en mis sistemas. Incluyen ajustes para shells (zsh), alias, funciones y configuraciones para facilitar y optimizar el trabajo en ciberseguridad, bug bounty y administración de sistemas.

## ¿Qué contiene?

- Configuración de **Zsh** (temas, alias, funciones, prompt)
- Archivos de entorno para customizar workstations
- Scripts útiles para hackers éticos y devs
- Organización pensada para Windows 11 + WSL2 + Kali

## Instalación rápida

Clona el repo y copia o linkea los archivos según tu sistema. 
Ejemplo:
```
git clone https://github.com/theoffsecgirl/dotfiles.git
cd dotfiles
````
