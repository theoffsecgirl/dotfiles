# hunt-ai templates

Cada fichero corresponde a un subcomando de `hunt-ai`.

- `analyze.md`: priorización de superficie existente.
- `hypotheses.md`: hipótesis limitadas, justificadas y comprobables.
- `caido.md`: análisis del historial mediante MCP en modo solo lectura.
- `report.md`: borrador de reporte condicionado a evidencia validada.

Las plantillas usan `{{TARGET}}`, sustituido por el script en tiempo de ejecución.

No incluyas secretos, tokens ni datos personales en estas plantillas. Las reglas comunes deben mantenerse breves y específicas; evita copiar prompts completos entre modos.
